# openclaw-dna

OpenClaw plugin for [agentic-dna](https://github.com/exisz/agentic-dna) — the agent governance system.

## Install

```bash
openclaw plugins install openclaw-dna
```

## What's included

- **DNA CLI commands** — `dna spec`, `dna philosophy`, `dna convention`, `dna workflow`, `dna cron`, and more
- **`imperial-dna` skill** — governance skill auto-registered with OpenClaw
- **Adaptive cron** — dynamic frequency adjustment for scheduled tasks

## Docs

See the [agentic-dna repo](https://github.com/exisz/agentic-dna) for full documentation.
