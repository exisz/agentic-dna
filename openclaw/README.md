# openclaw-dna

OpenClaw plugin for [agentic-dna](https://github.com/exisz/agentic-dna) — the agent governance system.

## Install

```bash
openclaw plugins install openclaw-dna
```

## What's included

- **DNA CLI commands** — `dna spec`, `dna philosophy`, `dna convention`, `dna protocol`, `dna flow`, `dna cron`, and more
- **`dna` skill** — governance skill auto-registered with OpenClaw
